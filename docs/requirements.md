# AI 英语口语陪练 — 需求文档

## 1. 项目概述

一款基于 AI 的英语口语练习工具，帮助用户在指定场景下进行真实对话训练。系统通过语音交互实现沉浸式口语练习，并提供发音评测、语法纠错与课后总结功能。

## 2. 用户角色

| 角色 | 描述 |
|------|------|
| 学习者 | 使用系统进行英语口语练习的主要用户 |

## 3. 功能需求

### 3.1 场景选择（P0）
- 支持至少 3 个预设场景：面试、点餐、商务会议
- 每个场景有独立的角色设定和对话上下文
- 用户可在开始对话前自由切换场景

### 3.2 实时语音对话（P0）
- 用户通过麦克风输入语音
- 系统实时识别并转写为文本
- AI 根据场景上下文生成自然回复
- 系统将回复合成为语音播放
- 支持打断机制：用户在 AI 讲话时可插话

### 3.3 发音评测（P1）
- 对用户语音进行发音准确性评分
- 标注发音错误的单词/音素
- 提供改进建议

### 3.4 语法/表达纠错（P1）
- 实时检测用户表达中的语法错误
- 区分严重级别：致命错误当场提示，轻微错误课后总结
- 提供正确表达建议

### 3.5 课后总结报告（P2）
- 生成本次练习的综合评分报告
- 包含：流利度、准确度、发音、词汇丰富度
- 列出高频错误和改进建议
- 与历史记录对比展示进步趋势

## 4. 非功能需求

| 需求 | 指标 |
|------|------|
| 端到端延迟 | ASR → LLM → TTS 全链路 ≤ 2 秒 |
| 语音识别准确率 | 英文 ≥ 90%（安静环境） |
| 并发支持 | 单用户单会话，后续可扩展 |
| 可用性 | 本地运行，无需联网即可使用核心功能 |

## 5. 技术约束

- 开发语言：Python 3.11
- 深度学习框架：PyTorch + CUDA 12.6
- GPU：NVIDIA RTX 4060 Laptop
- LLM API：DeepSeek（对话生成 + 语法纠错）
- ASR：faster-whisper（本地推理）
- TTS：edge-tts（免费高质量）
- 后端框架：FastAPI + WebSocket

## 6. 验收标准

- [ ] 用户可选择场景并开始对话
- [ ] 语音输入后 2 秒内获得语音回复
- [ ] 对话内容与场景高度相关
- [ ] 支持打断 AI 讲话
- [ ] 课后生成总结报告
